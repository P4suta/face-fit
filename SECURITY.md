# Security Policy

## Supported Versions

Security fixes are provided for the latest release.

| Version | Supported |
|---|---|
| 0.1.x | ✅ |

## Reporting a Vulnerability

Please report vulnerabilities privately via
[GitHub Security Advisories](https://github.com/P4suta/face-fit/security/advisories/new)
rather than a public issue. Where possible, include reproduction steps, impact,
and any suggested fix.

We aim to acknowledge reports within a reasonable time and will keep you updated
on the expected fix.
